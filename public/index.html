Planning Poker for Slack

A simple, no frills planning poker bot for agile estimation of team tickets and issues

Estimate Together, Decide Together

Planning Poker helps your team make better estimates by avoiding anchoring bias and encouraging discussion.

How It Works

1

Start a Session

Type /poker [issue or description] in any channel to start a new planning poker session.

2

Team Votes

Team members anonymously select their estimates (1, 2, 3, 5, or 8).

3

Reveal Results

Type /poker-reveal to show everyone's votes and discuss the results.

Features

Anonymous Voting

Prevent anchoring bias by keeping votes hidden until reveal.

Simple Interface

No complex setup or configuration required.

Interactive Buttons

Easy one-click voting with persistent buttons.

Vote Updates

Change your mind? Simply vote again to update.

Privacy Policy

Last updated: July 23, 2025

Information We Collect

Slack Planning Poker collects minimal information necessary for the app to function:

  • Slack workspace ID and access tokens (for authentication)
  • Channel IDs where the app is used
  • User IDs for those participating in voting sessions
  • Voting data (associated with user IDs)

How We Use Information

We use the collected information solely for:

  • Authenticating with the Slack API
  • Managing planning poker sessions
  • Recording and displaying votes

Data Retention

Voting data is retained for 30 days after a planning poker session concludes. After this period, the data is automatically deleted from our systems.

Third-Party Sharing

We do not share any collected information with third parties.
